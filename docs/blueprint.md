# **App Name**: RecipeWise

## Core Features:

- Recipe Search: Users can search for recipes by ingredient, cuisine (e.g., Italian, Mexican), or meal type (e.g., breakfast, dessert).
- Ingredient List Display: The application will show a complete list of ingredients with precise quantities for each recipe.
- Step-by-Step Instructions: Cooking instructions will be displayed in a clear, easy-to-follow, step-by-step format.
- Dietary Filters: Users can filter recipes to match specific dietary needs, including vegan, keto, and gluten-free options.
- Save Favorites: Registered users can save their favorite recipes to a personal collection for easy access.
- Shopping List Generator: Automatically creates a consolidated shopping list based on the ingredients from one or more selected recipes.
- Ratings & Reviews: Users can rate recipes and leave comments to share their feedback with the community.
- Meal Planner: Presents the user with three daily recipe suggestions (breakfast, lunch, and dinner) from their saved favorites or general catalog.
- Macros Breakdown: Each recipe card will display a nutritional breakdown, including macros (proteins, fats, carbohydrates).
- AI - Recipe Recommendations: The system will suggest similar recipes to the user based on their saved favorites, search history, and ingredient lists.
- AI - Ingredient Substitutions (LLM): Users can get suggestions for alternative ingredients if they are missing an item, powered by a Large Language Model.

## Style Guidelines:

- Primary color: Saturated yellow (#FFC107) to evoke a sense of appetite and culinary delight.
- Background color: Desaturated light-yellow (#FAF9F0) to provide a clean, appetizing backdrop.
- Accent color: Analogous orange (#FF9800) to highlight key interactive elements.
- Headline font: 'Playfair', serif. Body text: 'PT Sans', sans-serif. Use Playfair for headers and short descriptions and PT Sans for longer form recipe text.
- Use simple, outline-style icons for categories and filters.
- Subtle transitions when filtering recipes or adding items to the shopping list.
